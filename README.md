# ⚓ Eugen | Sailor Piece ⚓

> **"Elegansi adalah satu-satunya keindahan yang tidak pernah pudar."**
> — *Prinz Eugen*

Selamat datang di repositori resmi **Eugen | Sailor Piece**, asisten taktis tingkat lanjut yang dirancang khusus untuk efisiensi maksimal dalam pertempuran dan manajemen bos.

---

## 🚀 Fitur Utama

* **About Section**: Informasi lengkap mengenai versi, pemilik, dan status server saat ini.
* **Tactical Farm**: 
    * *Auto Basic Attack*: Serangan cepat tanpa henti.
    * *Multi-Select Auto Skill*: Pilih kombinasi skill (Z, X, C, V) yang ingin dieksekusi secara otomatis.
* **Overlord Spawner**: 
    * Mendukung pemanggilan bos **Atomic**, **Anos**, dan **Rimuru**.
    * Pengaturan tingkat kesulitan dinamis (Normal hingga Extreme).
    * *Supreme Auto Spawn*: Memastikan target selalu muncul di hadapanmu.
* **Persistent UI**: Tombol melayang "EUGEN" yang memastikan akses menu tidak hilang saat diminimalkan.
